# ERROR 1 — Ruff UP047

You saw:

```text id="jlwmvi"
Generic function should use type parameters
```

---

# WHY IT HAPPENED

Python 3.12 introduced:

* PEP 695
* new generic syntax

Old style:

```python id="jlwmvj"
T = TypeVar("T")
```

New style:

```python id="jlwmvk"
def func[T]()
```

Ruff enforced modern standards.

---

# HOW WE SOLVED IT

We modernized the generic syntax.

This improved:

* readability
* maintainability
* compliance with modern Python

---

# PROFESSIONAL LESSON

Tooling evolves with language evolution.

Good engineers:

* adapt
* modernize
* follow ecosystem best practices

---

# ERROR 2 — Missing YAML Type Stubs

You saw:

```text id="jlwmvl"
Library stubs not installed for "yaml"
```

---

# WHY IT HAPPENED

PyYAML:

* existed before Python typing became common
* lacks built-in typing support

mypy strict mode rejected untyped imports.

---

# HOW WE SOLVED IT

Installed:

```bash id="jlwmvm"
types-PyYAML
```

This provided:

* type definitions only
* no runtime changes

---

# PROFESSIONAL LESSON

Static typing ecosystems often require:

* stub packages
* typing adapters
* gradual migration support

Very common in real projects.
