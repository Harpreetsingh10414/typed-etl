# Interview Questions You Can Answer

### Why use Polars instead of Pandas?

Polars is faster, more memory efficient, multi-threaded, and increasingly adopted in modern Data Engineering workloads.

---

### Why create a unified file reader?

To eliminate duplicated file handling logic and provide a consistent API regardless of file format.

---

### Why return a single DataFrame type?

Consistency. Users should not need to care whether the source was CSV, JSON, or Parquet.

---

### Why log file reads?

To improve observability and simplify debugging in production environments.

---

### Why separate UnsupportedFileFormatError and FileReadError?

One represents an unsupported input type, the other represents a failure while processing a supported type.
